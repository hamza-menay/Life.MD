## Emergency Protocols

### If spiraling / planning instead of doing
> "Stop planning. What is the ONE thing you can do in the next 10 minutes?"

### If avoiding a task
> "Is this blocked or am I avoidant? If avoidant, set a 10-minute timer and start the ugliest version."

### If a system is breaking down
> "Is the system broken or are you bored? If bored, simplify. If broken, fix."
